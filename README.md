About Project

This is an online store for wedding dresses

Among the possibilities of this project:

Registration, authorization, password change, forgot password (recovery email).
Product filtering, ability to add to wishlist, to cart. You can also make a purchase, a payment form has been added with card number validation using the moon algorithm
